# Frontend — React SPA

Frontend-часть курсового проекта **«Платформа для онлайн-опросов и анкетирования»**.

## Стек
- React
- Vite
- React Router DOM
- Axios

## Скрипты

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Запуск

```bash
npm install
npm run dev
```

Frontend будет доступен на `http://localhost:3000/`.

## Важное

Во время разработки должен быть запущен backend на `http://127.0.0.1:8000/`.

Vite настроен с proxy для `/api`, поэтому запросы из SPA перенаправляются на Django backend.

## Возможности SPA

- регистрация и вход
- просмотр списка опросов
- фильтрация и поиск
- просмотр деталей опроса
- прохождение опроса
- комментарии
- создание опроса
- просмотр своих опросов
- просмотр статистики
- редак��ирование профиля
